# 🏥 Healthcare Waiting Time Dashboard

## 📌 Overview

This project is a Healthcare Dashboard designed to analyze and visualize patient waiting times. It provides insights into both **inpatient** and **outpatient** waiting durations using key statistical measures like **average** and **median**.

The dashboard also breaks down waiting times based on different **medical specialties**, helping stakeholders identify bottlenecks and improve service efficiency.

---

## 🚀 Features

* 📊 Displays **average waiting time**
* 📊 Displays **median waiting time**
* 🏥 Covers both **inpatient** and **outpatient** data
* 🔍 **Specialty-wise analysis** of waiting times
* 📈 Easy-to-understand visualizations
